# Meu-Homelab-p-s-apocal-ptico
Histórico de criação do meu homelab utilizando "sucata e lixo eletronico"

## Objetivo
Criar meu homelab e criar minha rede local sem maiores gastos com equipamentos, tentando utilizar hardwares pouco convencionais.

### Recursos que tenho até o dia de hoje 22-07-2026.
Notebook Samsung Ativ 370ek4, sendo utilizado como computador de uso diário, com o Sistema Operacional : linux mint 22-3 xfce, com processador celeron 3205u "muito ruim" com 4gb de ram e 500 gb de armazenamento.

minipc cce que preciso conferir o modelo.

projetor que preciso conferir o modelo

3 tvbox genericas que tambem preciso conferir o modelo

2 roteadores que preciso conferir o modelo

1 gpon da tplink que preciso conferir o modelo

até o momento o notebook é a unica coisa que configurei por enquanto, por ter o adquirido recentemente apenas troquei a s.o para o mint e estou configurando com base em meu uso pessoal, apenas para navegar e utilizar programas leves como o libre office.

apos conferir os outros aparelhos eu irei atualizar os modens com o OpenWRT
considero usar o desktop como servidor utilizando o casaOS, e os tvbox tenho pensado em utilizar 1 deles com o pihole, outro com o pfsense sendo utilizado apos substutuir o hgu da provedora pelo gpon, ele ira ser utilizado apenas como firewall e em conjunto com os roteadores para pontos de acesso, e o ultimo tvbox penso em utilizar com antenas de radio para coleta de informações como clima, trafego aereo etc, apenas por hobby mesmo, o projetor sera usado com o casaos para fazer a stream de filmes, series etc.
